01

Brand Identity

Brand Name

SHIFO

"Shifo" means "healing" in Tajik/Uzbek, reflecting the core mission of the platform: to bring modern digital solutions to healthcare management across Central Asia.

Mission Statement

To empower medical clinics with an intuitive, comprehensive, and secure digital management platform that streamlines operations, enhances patient care, and drives data-informed decision making.

Brand Values

Trust & Security

Patient data protection is paramount. Every feature is built with security first.

Simplicity

Complex healthcare workflows made intuitive and accessible for all staff levels.

Reliability

24/7 system availability. Clinics depend on us for uninterrupted operations.

Innovation

Continuously evolving with modern technology to serve healthcare better.

08

Iconography

Lucide icon library for consistent, clean line-based icons.

Icon Library

Shifo uses Lucide (lucide-angular). Lucide provides consistent, modern line-based icons that align with the brand aesthetic. Default stroke width is 2px, use 1.5px for large decorative icons (32px+).

Style Rules

Icons inherit text color by default
Always vertically center with adjacent text
8px minimum gap between icon and text
Use brand colors for emphasis only

Icon Sizing

Size Pixels Usage
xs 14px Inline with small text, badges
sm 16px Buttons, body text icons
md 20px Navigation, list icons
lg 24px Section headers, feature cards
xl 32px Feature showcases, empty states
2xl 48px Hero sections, branding
09

Voice & Tone

Professional yet approachable, reflecting the seriousness of healthcare with warmth.

Clear

Use plain language. Avoid jargon. Technical terms are explained when necessary.

Confident

Speak with authority. The platform is reliable, and the language reflects that.

Caring

Healthcare is personal. Words show empathy while maintaining professionalism.

Concise

Respect the reader's time. Say more with fewer words. Every word earns its place.

Multilingual Support

Shifo supports Russian and English as primary languages, with content localized for the Central Asian market. All UI text is translation-ready and stored in structured translation objects.
